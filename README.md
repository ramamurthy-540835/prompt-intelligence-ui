# AI Data Agents Platform

## Overview
This project is a GenAI-powered data engineering and orchestration platform built for automating data pipelines, agent workflows, and analytics on Google Cloud Platform (GCP).

## Project Structure

### `ai-data-agents/`
- **core platform code**
- **agents** → individual AI/data agents
- **orchestration** → workflow coordination
- **tools** → utilities and helpers
- **config** → configuration

### `data/`
- datasets or sample data

### `tests/`
- validation and testing

### `deploy/`
- deployment scripts

### `docs/`
- BUILD_PROCESS.md, DEMO_GUIDE.md, etc.

## Tech Stack
- **Python**
- **GenAI agents** (LLM-driven workflows)
- **GCP services** (BigQuery, Vertex AI, Cloud Run)
- **Orchestration framework**

## Key Features
- AI agents for data engineering
- Pipeline automation
- Orchestration of workflows
- Integration with GCP data services

## Architecture
- **Data Flow**: Data Sources → Agents → Orchestration → Storage (BigQuery)
- **AI Layer**: Decision making and automation
